Lalafo Parser Service
Это тестовое задание: сервис на Spring Boot, который берет данные с Lalafo и отдает их в чистом JSON.

Что делает проект:
Стучится в API lalafo.kg.

Забирает 100 актуальных объявлений.

Вытаскивает только нужное: название, цену, город, ссылку на фото и дату.

Отдает всё это по адресу http://localhost:9090/test-result.

Как я это реализовал:
Java 17 и Spring Boot.

Использовал Records вместо обычных классов (так короче и удобнее).

Настроил RestTemplate через бины.

Вынес настройки (ссылку на API и количество страниц) в файл application.yaml.

Написал AdMapper, чтобы превращать сложный ответ от Лалафо в мой простой формат.

Добавил специальные заголовки (headers), чтобы Лалафо не блокировал запросы.

Как запустить:
Собрать проект (создастся JAR):

Bash

./mvnw clean package
Запустить сам файл:

Bash

java -jar target/Kanat_Ergeshmamet_uulu.jar
Перейти в браузере сюда: http://localhost:9090/test-result
